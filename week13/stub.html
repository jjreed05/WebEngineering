AngularJS : Checklist Demo

Background on AngularJS

AngularJS was first introuduced in 2012. In 2009, two Google employees named Misko Hevery and Adam Abrons started to develop AngularJS as a side project. They envisioned their project to be an end to end tool that allowed web designers to interact with both the front-end and the back-end. Google ended up taking on AngularJS as a new project after Hevery show cased his project to his manager.AngularJS is a structural framework for dynamic web applications. It is primarily used for front-end web application framework. This framework is written in JavaScript. It is primarily a client-side web tool.

This Assignment

In this assingment, I found out about AngularJS through the research that I did for the Week 13 discussion board. I decided that I wanted create a demonstration for it. So, I created a checklist with AngularJS. I used an input box to get the user's input. It then takes that input and pushes it into a ul list of items. The user can then clear the checklist by clicking the checkbox near the item and clicking the completed button. Additionally, I created a stylesheet of my own and also used a template off of W3 Schools to try out.

The Checklist

Enter the items that you need to get done into the input box. Click enter for each item.

If you want to clear the completed items, click the check box near the item and click the "Item Completed" button
